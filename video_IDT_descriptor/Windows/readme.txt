Note: This code is a Win OS implemenation of improved DT feature based on Wang's Linux code on http://lear.inrialpes.fr/people/wang/improved_trajectories.

We use this code for IDT features in our paper of "Bag of Visual Words and Fusion Methods for Action Recognition: Comprehensive Study and Good Practice". We only provide the exe file by now.


### Usage ###

Just run the demo.m first.If you fail to use DenseTrack.exe, please try to install the followings in your system:
* Microsoft Visual Studio (tested with VS 2010)
* OpenCV library (tested with OpenCV-2.4.3)

If you have any problems or find any bugs, please free to contact me at xiaojiangp@gmail.com.

Enjoy!
